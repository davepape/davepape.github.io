CSS

HTML == information

CSS == presentation







Styling

Applies to elements; can control:

  • Colors - foreground, background, border
  • Text formatting - typeface, size, style, etc
  • Layout - position, layering
  • other stuff...







Example

<p style="font-family: Papyrus;">Lorem ipsum</p>

Yields:

Lorem ipsum







Rules

property: value;

For example 

color: red;
font-size: 20px;
display: block;






Where to place rules

  • HTML tag (inline)
  • <style> in header
  • CSS file (use <link> in header)






Selectors

Select what tag(s) to apply a ruleset to

  • HTML tag
  • element id (#)
  • element class (.)
  • other options (combinations, state, ...?)
